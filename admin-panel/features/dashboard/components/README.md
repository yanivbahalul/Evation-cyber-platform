# Dashboard Components

> **Owner:** Yaniv

The UI widgets that make up the live monitor.

| Component | Shows |
|-----------|-------|
| `ThreatMap` / `LeafletMap` | Attack origins on a world map |
| `AttackEventsTable` | Live feed of trap triggers |
| `AttackerProfiles` | Per-IP intel cards |
| `HoneyTokenPanel` | Issued and used honey tokens |
| `Sidebar` / `TopBar` | Navigation and global stats |

All of these render data pushed live from Max's telemetry.
